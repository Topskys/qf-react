# qf-app-dva
## DvaJS
dva 首先是一个基于 redux 和 redux-saga 的数据流方案（阿里），然后为了简化开发体验，dva 还额外内置了 react-router 和 fetch，所以也可以理解为一个轻量级的应用框架。https://dvajs.com/
```
$ npm install dva-cli -g
$ dva -v
# new project
$ dva new named
# 启动
$ cd named
$ npm start
```
